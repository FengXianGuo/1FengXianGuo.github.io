这里有两个属性Ancestors和FrameWork Listeners

Ancestors属性：

上图中：还是之前的测试用例，将Ancestors选项关闭。
会发现click的信息中，只剩下了div#test节点自身的事件。
不难看出：Ancestors主要用于分析祖先元素的事件

FrameWork Listeners属性：

上图中：关闭了FrameWork Listeners，会发现绑定事件的具体代码指向了jquery，
其实这没有错，事实上使用jquery绑定事件，就是jquery内部帮我们做了处理，但这样很不利于平时的开发和调试。

Framework listeners 勾上后浏览器会处理主流框架的绑定事件，这样事件绑定显示位置更加准确。
